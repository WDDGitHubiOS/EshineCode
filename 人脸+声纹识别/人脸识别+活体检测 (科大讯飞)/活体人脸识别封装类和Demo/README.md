# FaceRecognition
活体人脸识别，识别人脸，识别张嘴和摇头动作，并拍照，分拍照版和截图版，截图版没有声音，可以偷偷的拍照哦

这是在讯飞人脸识别第三方的基础上做的二次开发，在只识别人脸的基础上判断添加了识别人脸动作中的张嘴和摇头。
